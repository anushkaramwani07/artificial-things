# All artificial things are designed
Student-contributed design themes for an adapted, aesthetically edited excerpt from Chapter 1 of "The Design of Everyday Things," the book by Donald Norman.

Given identical HTML, students can contribute a new CSS file and JavaScript file to modify the page, hacking the Document Object Model with JS if desired.

View current themes at https://tordevries.github.io/artificial-things/.

Students: see the assignment in Canvas for details on how to contribute. In short: fork it, create a new directory with your files, commit it, then pull request.
